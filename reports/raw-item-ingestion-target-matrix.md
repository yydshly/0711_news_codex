# RawItem Ingestion v1 Target Matrix

Audit snapshot: 2026-07-12. This is a catalog audit, not a statement that every
target has passed three live fetch rounds. Values below are transcribed from the
reviewed source YAML files. “Risk” is the YAML total in the fixed order
`terms + authentication + stability + data_quality + operating_cost`; it is not a
legal conclusion. “Missing schema fields” means fields not required by that
target's `expected_fields`, not a claim that a provider can never supply them.

All listed access URLs are HTTPS. Every target has an explicit audited fallback:
“no additional audited method”. HTML-only catalog targets are **not** eligible
for automatic fetches: their YAML requires manual approval and their catalog
coverage does not imply that content is being read.
